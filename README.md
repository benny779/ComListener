# ComListener

Connect to serial devices, send and recive answer.\
I'm using [RJCP.IO.Ports](https://www-0.nuget.org/packages/SerialPortStream/) class to support Usb-Serial devices.

As of today, this project support [H650 Baby and Neonatal Scales](https://www.shekelonline.co.il/wp-content/uploads/HW-650.pdf) only.
